# Кейсы взаимодействия с ИИ-ассистентом управления задачами

## 🎯 Обзор
ИИ-ассистент должен вести себя как живой помощник, понимать естественную речь и активно помогать пользователю в управлении задачами.

## 📱 WhatsApp взаимодействия

### 1. Создание задач через WhatsApp

#### Кейс 1.1: Простая задача с дедлайном
```
Пользователь: "Напомни завтра в 10:00 позвонить Айдару"

Ассистент: "✅ Задача создана: Позвонить Айдару
📅 Завтра в 10:00
🔄 Напомню за 30 минут"
```

#### Кейс 1.2: Задача с относительным сроком
```
Пользователь: "До пятницы подготовить коммерческое предложение для клиента X"

Ассистент: "✅ Принято: Подготовить коммерческое предложение для клиента X
📅 Срок: пятница (через 3 дня)
⚡ Приоритет: высокий (судя по формулировке)"
```

#### Кейс 1.3: Задача после встречи
```
Пользователь: "После встречи с NCOC нужно отправить summary и список следующих шагов"

Ассистент: "✅ Задача создана: Отправить summary и список следующих шагов после встречи с NCOC
📅 Срок: без точного времени (привяжется к встрече в календаре)
🔗 Тип: follow-up по встрече"
```

#### Кейс 1.4: Задача с планированием времени
```
Пользователь: "На следующей неделе выделить 2 часа на бюджетирование"

Ассистент: "✅ Запланировано: Выделить 2 часа на бюджетирование
📅 Следующая неделя
⏰ Длительность: 2 часа
💡 Рекомендую вторник 14:00-16:00 (свободное время в календаре)"
```

### 2. Запросы информации

#### Кейс 2.1: Повестка на сегодня
```
Пользователь: "Что у меня сегодня?"

Ассистент: "📋 Ваш день сегодня:

🌅 Утро:
• 9:00-10:00: Встреча с командой разработки
• 10:30: Позвонить Айдару (напоминание)

📝 Задачи на сегодня:
✅ Подготовить отчет по проекту (выполнить до 12:00)
⏳ Ответить на письмо от клиента (важно)

🚨 Просрочено:
❌ Отправить договор (было вчера)

💡 Свободные окна:
• 14:00-15:00: Можно выполнить задачу бюджетирования
• 16:00-17:00: Резерв для срочных дел"
```

#### Кейс 2.2: Список задач
```
Пользователь: "Какие у меня задачи?"

Ассистент: "📝 Ваши активные задачи:

🔥 Высокий приоритет:
• Подготовить коммерческое предложение для клиента X (до пятницы)

⚡ Средний приоритет:
• Позвонить Айдару (завтра 10:00)
• Ответить на письмо от партнера (до среды)

📋 Низкий приоритет:
• Изучить новый инструмент (без срока)

Всего: 4 активных задачи"
```

#### Кейс 2.3: План на неделю
```
Пользователь: "План на неделю"

Ассистент: "📅 План на неделю (22-28 апреля):

📊 Загруженность:
• Понедельник: 80% (3 встречи)
• Вторник: 40% (рекомендую для глубокой работы)
• Среда: 90% (встречи с клиентами)

🎯 Ключевые задачи:
• Сегодня: Подготовить отчет по проекту
• Завтра: Позвонить Айдару
• До пятницы: Коммерческое предложение для клиента X

⚠️ Риски:
• Среда перегружена - перенести неважные встречи
• 2 просроченные задачи требуют внимания

💡 Рекомендации:
• Выделить вторник для стратегических задач
• Не забыть follow-up по встрече с NCOC"
```

```
Пользователь: "Отметил отчет выполненным"

🎉 Отлично! У вас осталось 3 активных задачи."
```

#### Кейс 3.2: Изменить срок
```
Пользователь: "Перенеси звонок Айдару на завтра 11:00"

Ассистент: "✅ Время изменено: Позвонить Айдару
📅 Новый срок: завтра в 11:00
🔄 Напоминание обновлено"
```

#### Кейс 3.3: Изменить приоритет
```
Пользователь: "Сделай бюджетирование высоким приоритетом"

Ассистент: "⚡ Приоритет повышен: Выделить 2 часа на бюджетирование
Теперь в списке высоких приоритетов"
```

### 4. Уточняющие вопросы

#### Кейс 4.1: Неясный срок
```
Пользователь: "Нужно подготовить презентацию"

Ассистент: "📝 Понял задачу: Подготовить презентацию

❓ Уточните, пожалуйста:
• К какому сроку нужна презентация?
• Для какой встречи/проекта?

💡 Например: 'До конца недели' или 'К понедельнику'"
```

#### Кейс 4.2: Неоднозначный контекст
```
Пользователь: "Не забыть про отчет"

Ассистент: "📝 Задача зафиксирована: Не забыть про отчет

❓ Что именно за отчет?
• Еженедельный отчет по проекту?
• Отчет для клиента?
• Финансовый отчет?

💡 Уточните, чтобы поставить правильный приоритет и срок"
```

## 📧 Email взаимодействия

### 5. Пересылка писем

#### Кейс 5.1: Письмо с комментарием
```
Тема: FWD: Срочный запрос от клиента

Текст письма:
"Нужно срочно подготовить ответ до четверга"

Пересланное письмо: [содержимое от клиента]

Ассистент: "📧 Письмо обработано!

✅ Задача создана: Подготовить ответ на запрос клиента
📅 Срок: четверг (через 2 дня)
⚡ Приоритет: высокий (указано 'срочно')

💡 Источник: письмо от клиента [ссылка]
🔗 Привязан к email thread"
```

#### Кейс 5.2: Письмо без явного дедлайна
```
Тема: FWD: Обсудить бюджет на следующий квартал

Текст: "Проконтролировать и подготовить предложения"

Ассистент: "📧 Задача зафиксирована: Проконтролировать бюджет на следующий квартал и подготовить предложения

❓ Уточните срок выполнения:
• До конца недели?
• К следующему понедельнику?
• Без спешки?

💡 Без срока задача попадет в 'когда-нибудь'"
```

## 📅 Календарь интеграция

### 6. Учет занятости

#### Кейс 6.1: Предложение свободного времени
```
Пользователь: "Запланируй созвон с командой на эту неделю"

Ассистент: "📅 Анализ календаря...

✅ Предлагаю время: вторник 14:00
• Свободно 1 час
• После встречи с клиентом
• До обеденного перерыва

Альтернативы:
• Четверг 16:00 (45 мин свободно)
• Пятница 11:00 (1.5 часа свободно)

❓ Подходит вторник 14:00?"
```

#### Кейс 6.2: Предупреждение о конфликте
```
Пользователь: "Запланируй встречу с поставщиками на завтра 10:00"

Ассистент: "⚠️ Конфликт времени!

📅 У вас уже запланировано:
• 10:00-11:00: Встреча с командой разработки

💡 Альтернативные варианты:
• Завтра 14:00 (свободно)
• Послезавтра 10:00 (свободно)
• Через неделю вторник 10:00

❓ Перенести существующую встречу или выбрать другое время?"
```


## 🎯 Ключевые принципы взаимодействия

1. **Естественность** - отвечать как живой помощник
2. **Проактивность** - предлагать варианты и предупреждать
3. **Эмпатия** - учитывать эмоциональное состояние пользователя
4. **Контекст** - помнить предыдущие взаимодействия
5. **Действенность** - не болтать, а решать проблемы
6. **Гибкость** - адаптироваться под стиль пользователя

## 📊 Метрики успеха

- **Вовлеченность**: Пользователь пишет регулярно
- **Эффективность**: Задачи выполняются в срок
- **Удовлетворенность**: Положительные отзывы
- **Автоматизация**: Минимум ручных действий
